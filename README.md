# MNTN Landing Page

A modern, responsive single-page landing website for **MNTN** — built with pure HTML, CSS

Designed with clean typography, smooth scrolling sections and attractive hero, features, about, and footer sections.

Live Demo:  
https://samcodes69-cpu.github.io/MNTN-landing-page/

## ✨ Features

- Fully responsive design (mobile, tablet, desktop)
- Smooth animations and hover effects
- Clean folder structure with separated CSS files
- Optimized file naming (herostyle.css, navstyle.css, etc.)
- Easy to customize colors, fonts, and content

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Media Queries)
